📖 Инструкция администратора

⚠ Данный раздел доступен только пользователям с ролью «Администратор».

Содержание

  1. Управление пользователями
  2. Загрузка пациентов из Excel
  3. Управление складами
  4. Управление ячейками
  5. Генерация QR-кодов для ячеек
  6. Просмотр полной истории операций

1. Управление пользователями

1
Перейдите в раздел «Пользователи»

В левом меню выберите «👥 Пользователи».

2
Создание пользователя

Нажмите кнопку «+ Добавить».

Заполните обязательные поля:

  • Логин — уникальный идентификатор (латиница, до 50 символов)
  • Пароль — не менее 6 символов
  • Полное имя — ФИО сотрудника
  • Профиль — выберите одну из ролей:
    • Администратор — полный доступ, управление пользователями и справочниками
    • Сотрудник склада — создание, размещение, перемещение, выдача описей
    • Приёмщик — только создание и выдача описей
3
Редактирование / блокировка

Нажмите на пользователя в таблице для редактирования.

Кнопка 🔒/🔓 блокирует/разблокирует учётную запись.

⚠ Важно: Заблокированный пользователь не сможет войти в систему.

2. Загрузка пациентов из Excel

1
Перейдите в раздел «Пациенты (Excel)»

В левом меню выберите «👥 Пациенты (Excel)».

2
Подготовьте Excel-файл

Файл должен быть в формате .xlsx со следующими колонками:

Номер историиФИОДата рождения
ИБ-10001Иванов Иван Иванович20.03.1980
ИБ-10002Петров Петр Петрович15.07.1975

Первая строка считается заголовком и пропускается.

3
Загрузите файл

Нажмите «Выберите файл», выберите Excel и нажмите «📥 Загрузить».

Система покажет результат: сколько загружено, сколько пропущено.

4
Экспорт (образец формата)

Нажмите «📤 Скачать Excel», чтобы получить текущий список всех пациентов.

3. Управление складами

1
Перейдите в раздел «Склады»

В левом меню выберите «🏠 Склады».

2
Создание склада

Нажмите «+ Добавить склад».

Укажите:

  • Наименование — название склада
  • Тип:
    • Склад приёмного отделения — не может содержать ячейки
    • Склад долговременного хранения — может содержать ячейки

4. Управление ячейками

1
Перейдите в раздел «Ячейки»

В левом меню выберите «📦 Ячейки».

2
Создание ячейки

Нажмите «+ Добавить ячейку».

  • Наименование — например: А-01, Б-03
  • Склад — только склады долговременного хранения
⚠ Ячейки можно создавать только на складах долговременного хранения.

5. Генерация QR-кодов для ячеек

1
В разделе «Ячейки» нажмите «🖨 QR»

Рядом с нужной ячейкой нажмите кнопку «🖨 QR».

2
Распечатайте QR-код

Откроется окно с QR-кодом. Распечатайте его и наклейте на ячейку.

При приёмке вещей сотрудник сможет отсканировать QR-код ячейки для быстрого выбора.

6. Просмотр полной истории операций

1
Перейдите в раздел «История операций»
2
Используйте расширенные фильтры

Администратор видит все операции всех пользователей.

Можно фильтровать по пациенту, сотруднику, дате.

3
Экспорт в Excel

Нажмите «📥 Экспорт» для выгрузки результатов в Excel.

По техническим вопросам обращайтесь к разработчикам системы
