1. Вход в систему

1
Откройте браузер

Запустите любой современный браузер (Google Chrome, Mozilla Firefox, Яндекс.Браузер, Microsoft Edge).

2
Перейдите по адресу системы

В адресной строке браузера введите:

https://itemstorage-front-ppak.mXX.dzm/

где XX — код вашей медицинской организации.

3
Авторизуйтесь в системе

В открывшейся форме введите:

  • Логин — ваш индивидуальный логин (например, IVANOVAA)
  • Пароль — ваш пароль

Нажмите кнопку «Войти».

4
Главная страница

После успешной авторизации откроется главная страница с разделами:

  • Поиск — поиск пациентов
  • Описи хранения — список всех описей
  • История операций — журнал действий
  • Словарь вещей — справочник наименований
  • Инструкция пользователя — текущий документ

В правом верхнем углу отображается ваше ФИО и кнопка «Выход».

2. Поиск пациента и создание описи

1
Перейдите в раздел «Поиск»

В левом меню выберите пункт «Поиск».

2
Найдите пациента

В поисковой строке введите один из критериев:

  • ФИО полностью или частично — например: Иванов или Иванов И.И.
  • Номер истории болезни — например: ИБ-10001
  • ФИО и дата рождения — например: Иванов И.И. 20.03.1980

Нажмите кнопку «🔍 Найти».

3
Выберите пациента

В результатах поиска нажмите кнопку «Добавить опись хранения» рядом с нужным пациентом.

4
Заполните опись вещей

В открывшемся окне:

  1. В поле «Название вещи» введите наименование (или выберите из словаря)
  2. В поле «Количество» выберите число
  3. В поле «Описание» укажите особые приметы: цвет, размер, состояние

Для добавления ещё одной вещи нажмите «+ Добавить вещь».

5
Сохраните опись

После заполнения всех вещей нажмите «Сохранить опись».

⚠ Важно: После сохранения редактирование описи будет недоступно.
6
Распечатайте бланк описи

Нажмите кнопку «📥 Скачать», чтобы получить печатную форму.

Проверьте поля:

  • ФИО Отдающего — пациент
  • ФИО Принимающего — сотрудник

Распечатанный бланк подписывается обеими сторонами.
